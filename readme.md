# Mess Management System V1.0

Built with Procedural PHP & Mysqli for one of my younger brother's univeristy project. Very simple & lightweight system which can be used by any person who want to manage his mess dynamically.

**Features of MMS 1.0**
* User Role
* User Management
* Date Wise Diposit System
* Date Wise Meal System
* Profile Update
* Active/Inactive User
* More to come if you want to contribute or share ideas.

## Getting Started

Clone the repository to your local machine or download the zip file to your pc to run.


## Built With

* [Skeleton](http://getskeleton.com/) - CSS Framework
* [jQuery](https://jquery.com/) - Used to generate datepicker & creating modal windows
* [DataTable](https://datatables.net/) - For showing data on tables
